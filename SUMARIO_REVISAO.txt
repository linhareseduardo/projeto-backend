🎯 RESUMO EXECUTIVO

  O projeto implementa uma API REST completa para um ecommerce de produtos 
  esportivos com as seguintes características:

  • Database com 6 tabelas (MySQL 8.0)
  • 16 endpoints REST com autenticação JWT
  • Auto-populate com 10 usuários + 5 categorias + 15 produtos realistas
  • 26 testes cobrindo todas as funcionalidades
  • Documentação completa para execução
  • Postman collection para demonstração
  • Deploy 100% reproduzível em qualquer máquina

  Tecnologias: Node.js, Express, Sequelize, MySQL, JWT, Jest
